Here's a breakdown of the technologies we use everyday that I love so dearly:

Operating Systems

In the beginning the lord made electricty. Then capicitors, logic gates and transistors... let there be light!

Often overlooked by those that love to code but of course our code needs somewhere to run free. Operating systems expose electrical hardware to programmable software via their amazing firmware. This is so developers don't need to write yucky binary machine code and can instead build applications with more readable programming languages such as C#.

They started out as digital text editors with basic memory and evolved into the full-spec graphical user interfaces we know and love today. New frontiers like VR push the boundaries of how we engage with computers but whether it's a screen or a hologram, we'll always need operating systems.

Web Browsers

Web browsers are arguably the most important piece of software installed on any operating system. They turn your lonely computer into an internet-connected machine with the collective knowledge and tools of humanity at your fingertips.

They're packed full of great features for web developers to play with and provide all the amazing gizmos needed to run websites and communicate to the outside world, such as: Code Interpretation, Graphical Rendering Engines, Network Code and Encryption Tools for secure communications between your website, your friends website and every other website in all the land.

HTML, CSS & JavaScript

The dream team. These 3 technologies are the heart and soul of every website and have fascinated me since I first explored the internet as a child. Think of HTML as the skeleton, JavaScript as the muscles and CSS as the skin.

For examples of HTML, CSS & JavaScript you need look no further than this very website.

The Extended Javascript Universe

Node.js & Deno

From humble beginnings as a DOM manipulator (DOM = Document Object Model - it's how a browser models a web page) to working with files and data on machines and microcontrollers. JavaScript has worked it's way into all areas of technology.

Facebook's React library links JavaScript variables to the DOM via "state" variables. It uses server-side JavaScript to generate optimised files for web pages and server-side render web content (SSR is a deeper concept - you'll have to get in touch with me personally if you want to find out more). Changes in state are first calculated in a Virtual DOM (VDOM) then efficiently mapped to the DOM via a process called VDOM-diffing. The React library resulted in a huge productivity boost for frontend web developers and has therefore exploded in popularity. It's spinoff cousin React Native brings the same system to mobile application development and Electron brings web applications to desktop by utlising Google's open-source Chromium engine - awesome!

How does JavaScript move beyond frontend user applications? Node.js (made with C++) or it's successor Deno (made with Rust), that's how. Both powerful engines for running Javascript outside of web browsers, with additional functionality for executing UNIX commands and working with files. NodeJS is wildly popular and drives the vast majority of open-source code today!

The Low-Level Powerhouses: C++ & Rust

Kickstarter of the age of information. C++ is everywhere. Your operating system? C++. Your plane's operating system? C++. The browser in which you are currently viewing this website? C. +. +. Created for ultimate flexibility by the hilariously quintessentially-nerdy Swedish genius, Bjarne Stroustrup. There's nothing this language can't do. It might just require a million lines of code to get there...

My relationship with C++ began at university, we met in the midst of confusing times. I was a 1st year BEng Aerospace Engineering student at the University of Bristol in the United Kingdom and was quickly realising that my course was going to involve a lot of complex analytical methods (good ol' pen, paper & a calculator engineering). Being the nerd I am I was delighted to see a computing module on my timetable. Sure enough, it was love at first sight. It was in those classes that I began to consider a career in programming and I never looked back.
