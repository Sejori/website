Here's a breakdown of the technologies we use everyday that I love so dearly:

Operating Systems

In the beginning the lord made electricty. Then logic gates, capicitors and transistors, then... let there be light!

Often overlooked by those that love to code but code needs somewhere to run free. Operating systems expose electrical hardware to applications via simpler programming languages such as C# so they don't need to be written in machine code.

They started out as digital text editors with basic memory and evolved into the full-spec graphical user interfaces we know and love today. New frontiers like VR push the boundaries of how we engage with computers more and more but screen or holograms, we'll always need operating systems.

Web Browsers

Web browsers are arguably the most important piece of software on any operating system. They turn your lonely computer into an inter-connected machine with the collective knowledge and tools of humanity at your fingertips.

They're packed full of great features for web developers to play with and provide all the amazing gizmos needed to run websites and communicate to the outside world, such as: Code Interpretation, Graphical Rendering Engines, Network Code and Encryption Tools for secure communications between your website, your friends website and every other website in all the land.

The Extended Javascript Universe

Node.js

From humble beginnings as a DOM manipulator (Document Object Model - it's how the browser interprets HTML & CSS) to working with files and data on machines and microcontrollers. JavaScript has worked it's way into all areas of technology.

Facebook's React library links your dynamic JavaScript variables to the DOM via it's "state" variables, a huge boost for people coding web application. It's spinoff cousin React-Native brings the same system to mobile application development and Electron brings web applications to desktop by utlising Google's open-source Chromium engine - awesome!

How does JavaScript move beyond frontend user applications? Node.js (or it's successor Deno), that's how. A powerful engine for running Javascript outside of web browsers, with additional functionality for executing UNIX commands and working with files. It's wildly popular and drives the vast majority of open-source code around today.

C++

Kickstarter of the age of information. C++ is everywhere. Your operating system? C++. Your plane's operating system? C++. The browser in which you are currently viewing this website? C. +. +. Created for ultimate flexibility by a hilariously nerdy-looking Swedish genius. There's nothing this language can't do. It might just require a million lines of code to get there...

My relationship with C++ began at university, we met in the midst of confusing times. I was a 1st year BEng Aerospace Engineering student at the University of Bristol in the United Kingdom and was quickly realising that my course was going to involve a lot of complex analytical methods (good ol' pen, paper & a calculator engineering). Being the nerd I am I was delighted to see a computing module on my timetable. Sure enough, it was love at first sight. It was in those classes that I began to consider a career in programming and I never looked back.

hint: a common exercise when learning about numerical methods in C++ is to write a cubic combination puzzle solver. I only got as far as the white cross before I got distracted writing web games...
